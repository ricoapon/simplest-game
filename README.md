# Simplest Game

The YouTube Video  [What is the simplest possible game](https://www.youtube.com/watch?v=A-lh1-bTzTw) talks about a game where to players can write `X` or `O` in a sequence. Each player has a separate goal of creating repeating patterns of a certain length.

This is a simple app in the browser that can be used to play the game.

## Technical

This is a static site that uses React (without build step). A simple test has been created in `game.test.js`. Most of the code has been generated by AI.
